# Only Aces Flagship Buildout Cost

This is a rough-order-of-magnitude buildout estimate for the first public-facing flagship location.

It is meant for internal planning, deck support, and early vendor outreach. It is **not** a bid package.

## Recommended budgeting posture

If the goal is a credible first flagship with:

- 3 to 4 climate-controlled hitting bays
- one 111-yard challenge shot
- commercial-grade safety containment
- automated ball dispensing
- camera verification
- lighting, networking, and electrical

then there are really two different public-site postures:

- **bootstrap flagship:** visibly real, customer-facing, but aggressively value-engineered
- **full flagship:** more permanent, more polished, and much less forgiving on cost

The prior version of this estimate leaned too far toward the second posture.

The cleaner working range for a first site is:

- **bootstrap public flagship:** `~$90k-$175k`
- **credible mid-case flagship:** `~$175k-$250k`
- **heavier full flagship:** `~$300k+`

For planning purposes, the best single underwriting number is now:

- **target bootstrap flagship budget:** `~$125k-$175k`

That assumes you deliberately avoid overbuilding the bays, keep the tech stack simple, and treat the first site as a brand-validating revenue machine rather than a finished chain prototype.

## Full flagship reference budget

If you instead want a more permanent, climate-controlled, polished first site, the earlier posture still applies:

- **minimum viable full flagship:** `~$200k`
- **credible base-case full flagship:** `~$330k`
- **more polished flagship:** `~$500k+`

## Low / base / high scenario view

| Scenario | Description | Total buildout |
| --- | --- | ---: |
| Bootstrap public flagship | presentable first site, aggressive value engineering | $90k-$175k |
| Mid-case public flagship | better finish level and more contingency | $175k-$250k |
| Full flagship | more permanent structure and infrastructure | $300k-$525k+ |

## Biggest cost drivers

These are the items most likely to move the number materially:

- netting height, span, and install complexity
- how polished the bay structure needs to feel
- trenching, conduit, and electrical distance from source power
- whether retrieval remains simple or becomes more engineered
- how much permitting or site engineering is required

## What this estimate excludes

This buildout view does **not** include:

- land acquisition or long-term ground lease economics
- ongoing rent, CAM, or property tax
- payroll before or after opening
- prize liability or promotional reserve
- POS/software subscriptions beyond light setup assumptions
- working capital for launch
- major restroom, parking, or utility upgrades outside the immediate build

## Simple investor-safe wording

If you need a clean sentence for a memo or pitch:

> We believe the first public-facing Only Aces site can likely be opened in roughly the `$125k-$175k` range if the build is kept intentionally simple, with cost moving materially higher if the team chooses a more permanent flagship structure, heavier sitework, or larger safety infrastructure.
