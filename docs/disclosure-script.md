# Private Disclosure Script

Use only in a serious partnership conversation. Not for deck copy, not for the site, and not for
first-contact messaging.

## Direct version

Chris, there’s one personal issue I want to put on the table early because I don’t ever want it to
surface later as a surprise. Years ago I got into legal trouble involving an underage girl and
served time in Texas. As a result, I have an ongoing lifetime registration obligation that is public.
I hate that this is part of my story, but it is, and I’d rather address it directly than let it
become a hidden issue. I also don’t want it to become a distraction from the business. If this
concept moves forward and you still see value in working together, I’m fully open to structuring my
role in a way that keeps me non-public while still allowing me to contribute meaningfully.

## Softer version

Chris, before anything got serious, there’s one background issue I’d want to disclose directly.
Years ago I got into legal trouble that resulted in prison time and a lifetime public registration
obligation in Texas. It’s public and searchable, so I don’t want it ever becoming a surprise later.
I’m telling you because I want to handle it cleanly and because, if necessary, I’m completely open
to a structure where I remain non-public while still helping build the concept.

## Delivery notes

- disclose only if there is real interest
- do not put this in written pitch materials
- do not hide it once the discussion becomes serious
- say it cleanly and stop talking
- let him react
